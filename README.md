# Pancreatic Cancer Detection using Machine Learning

Early detection of pancreatic cancer significantly improves patient prognosis and survival rates. This project leverages machine learning algorithms to analyze diagnostic data to accurately predict and classify the likelihood of pancreatic cancer.

## Key Features
* **Machine Learning Pipeline:** Implements supervised learning algorithms for classification.
* **Data Preprocessing:** Handles missing data and normalization to optimize performance.
* **Evaluation Metrics:** Metrics include Accuracy, Precision, and Recall to ensure reliability.

## Tech Stack
* **Language:** Python
* **Libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn
